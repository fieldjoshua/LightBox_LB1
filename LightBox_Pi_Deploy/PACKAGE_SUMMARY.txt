🚀 DEPLOYMENT OPTIONS:

OPTION 1 - Automated SD Card Setup:
1. Run: ./setup_sd_card.sh
2. Boot Pi and SSH in
3. Run: /boot/deploy_to_pi.sh

OPTION 2 - Manual SD Card Copy:
1. Copy files to SD boot partition
2. Boot Pi and SSH in  
3. Run deployment script

OPTION 3 - Direct Pi Copy:
1. scp files to existing Pi
2. Run: ./deploy_to_pi.sh
